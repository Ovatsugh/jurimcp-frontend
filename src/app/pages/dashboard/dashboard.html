Como Usar

01

Autentique com a API Key

Envie sua API Key no header X-Api-Key em todas as requisições ao MCP Server. Não é necessária troca de token. 

X-Api-Key: sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
02

Chame as tools via HTTP POST

Envie requisições JSON-RPC 2.0 para POST /mcp/jurimcp. Sem sessão, sem SSE — uma chamada HTTP por tool. 

# Python (pip install langchain-mcp-adapters)
from langchain_mcp_adapters.client import MultiServerMCPClient

client = MultiServerMCPClient({
    "jurimcp": {
        "url": "https://api.jurimcp.com/mcp/jurimcp",
        "transport": "streamable_http",
        "headers": { "X-Api-Key": "sk_SUA_CHAVE_AQUI" },
    }
})
tools = await client.get_tools()
