MCP Server

Guia de Uso

Ferramentas de cálculo jurídico expostas via JSON-RPC 2.0 sobre Streamable HTTP. Autentique com sua API Key e conecte qualquer cliente MCP compatível.

Visão Geral

Base URL https://api.jurimcp.com
Transport Streamable HTTP
MCP Endpoint POST /mcp/jurimcp
Autenticação X-Api-Key: sk_xxxx

Autenticação

Para acessar o MCP Server, use sua API Key diretamente no header X-Api-Key. Não é necessária nenhuma troca de token — a chave é usada diretamente em todas as requisições. Crie sua API Key no Dashboard.

Endpoint POST /mcp/jurimcp
Header X-Api-Key: sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Exemplo

API Key

Suas chaves começam com sk_. Cada chave dá acesso a todas as tools disponíveis no servidor.

Revogação

Se sua API Key for comprometida, revogue-a imediatamente no Dashboard e crie uma nova.

Status

Código Status Descrição
200 OK Requisição processada com sucesso.
400 Bad Request Parâmetros inválidos ou ausentes na requisição.
401 Unauthorized Token ausente, inválido ou expirado.
403 Forbidden Token válido, mas sem permissão para a tool solicitada.
404 Not Found Endpoint ou tool não encontrada.
422 Unprocessable Entity Argumentos da tool não passaram na validação.
429 Too Many Requests Rate limit excedido. Aguarde antes de tentar novamente.
500 Internal Server Error Erro inesperado no servidor.
